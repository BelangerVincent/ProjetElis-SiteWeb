Présentation du Toutou 3.0

Le Toutou 3.0 est un ourson en peluche innovant, capable d'interagir avec les enfants de manière naturelle et adaptée grâce à une combinaison de technologies de pointe :

  • Un Raspberry Pi 4 pour la puissance de calcul
  • Un microphone et un haut-parleur pour la communication audio
  • Une intelligence artificielle basée sur ChatGPT pour le traitement du langage naturel

Fonctionnement technique

Le Toutou 3.0 utilise un Raspberry Pi 4 comme unité centrale, sur lequel est installé un script Python qui gère l'interaction avec l'enfant. Voici les principales étapes du processus :

  1. Le microphone capte les paroles de l'enfant
  2. Le script Python convertit l'audio en texte grâce à une API de reconnaissance vocale
  3. Le texte est envoyé à l'API de ChatGPT pour interprétation et génération d'une réponse appropriée
  4. La réponse de ChatGPT est convertie en audio grâce à une API de synthèse vocale
  5. L'audio est joué via le haut-parleur du Toutou 3.0

Code source

Le code source du Toutou 3.0 est disponible sur GitHub à l'adresse suivante : https://github.com/PatBQc/ProjetElis

Il est écrit en Python et utilise plusieurs bibliothèques externes pour la reconnaissance vocale, la synthèse vocale et l'interaction avec l'API de ChatGPT.

Voici un aperçu simplifié de l'algorithme principal :

            Tant que l'enfant veut parler :
                Écouter les paroles de l'enfant via le microphone
                Convertir l'audio en texte
                Envoyer le texte à ChatGPT
                Recevoir la réponse de ChatGPT
                Convertir la réponse en audio
                Jouer l'audio via le haut-parleur

Défis et solutions

Lors du développement du Toutou 3.0, nous avons dû faire face à plusieurs défis, notamment :

  • La reconnaissance de la parole des enfants, parfois imprécise ou hésitante
  • La génération de réponses adaptées au niveau de langage et de compréhension des enfants
  • L'intégration de tous les composants dans un format de peluche agréable et sécurisé

Nous avons résolu ces défis en travaillant étroitement avec des experts en traitement du langage naturel, en psychologie de l'enfant et en design de produits. Nous avons également réalisé de nombreux tests utilisateurs pour affiner les réglages de notre système.

Évolution du projet

Le Toutou 3.0 est en constante évolution pour s'adapter aux besoins et aux retours des utilisateurs. Parmi les pistes d'amélioration envisagées, on peut citer :

  • L'ajout d'une caméra pour permettre une interaction visuelle et la reconnaissance des émotions
  • Le développement d'une application mobile compagnon pour suivre les progrès de l'enfant
  • L'intégration de nouveaux scénarios d'interaction (jeux éducatifs, histoires interactives, etc.)

Nous sommes ouverts à toutes les suggestions et idées pour faire du Toutou 3.0 un compagnon toujours plus enrichissant et stimulant pour les enfants.
